# Fonts Directory

This directory should contain the Neue Montreal font files.

## Required Font Files

Place the following font files in this directory:

- `NeueMontreal-Regular.otf` (weight: 400)
- `NeueMontreal-Medium.otf` (weight: 500)
- `NeueMontreal-Bold.otf` (weight: 700)

## Alternative Formats

If you have the fonts in different formats (`.woff`, `.woff2`, `.ttf`), you can update the paths in `src/app/layout.tsx` accordingly.

## Note

Neue Montreal is a commercial font. Make sure you have the proper license to use it in your project.
